# git branching

- `git branch` 현존 브랜치
- git branch [이름] 새로 만들기
- git switch (최신버전에서 추가) : 브랜치 바꾸기
- git checkout : 헤드나 브랜치 이동
- git checkout -b test : 만들고 바로 이동
- git switch -c test : 만들고 바로이동
